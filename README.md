# AgentLens

AgentLens is an AI agent debugging and cost tracking platform.

It helps developers understand what their AI agents are doing.

With AgentLens you can:

- Track agent executions
- See every execution step
- Monitor tool calls
- Find errors
- Measure execution time
- Track token usage
- Calculate request costs

## Tech Stack

- FastAPI
- PostgreSQL
- Redis
- Docker
- SQLAlchemy
- Pydantic
- uv
